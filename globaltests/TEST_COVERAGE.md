# Покрытие тестами

## Структура тестов

### 1. basic.spec.ts - Базовые тесты системы
- ✅ Загрузка главной страницы
- ✅ Проверка health check API
- ✅ Проверка root endpoint API
- ✅ Навигация между страницами
- ✅ Отображение header с логотипом

### 2. auth.spec.ts - Тесты аутентификации
- ✅ Регистрация нового пользователя
- ✅ Вход с валидными credentials
- ✅ Ошибка при неверных credentials
- ✅ Выход из системы
- ✅ Редирект на login при доступе к защищенным страницам
- ✅ Сохранение состояния авторизации после перезагрузки

### 3. items.spec.ts - Тесты управления товарами
- ✅ Создание нового товара
- ✅ Просмотр деталей товара
- ✅ Список товаров на главной странице
- ✅ Редактирование собственного товара
- ✅ Удаление собственного товара
- ✅ Отображение своих товаров в профиле

### 4. search.spec.ts - Тесты поиска и фильтрации
- ✅ Поиск товаров по текстовому запросу
- ✅ Фильтрация по категории
- ✅ Фильтрация по диапазону цен
- ✅ Сортировка товаров по цене
- ✅ Отображение пустого состояния при отсутствии результатов
- ✅ Навигация к деталям товара из результатов поиска

### 5. rentals.spec.ts - Тесты бронирований
- ✅ Создание заявки на аренду
- ✅ Просмотр списка бронирований
- ✅ Подтверждение заявки владельцем
- ✅ Отмена бронирования

### 6. profile.spec.ts - Тесты профиля пользователя
- ✅ Просмотр страницы профиля
- ✅ Обновление информации профиля
- ✅ Отображение товаров пользователя в профиле
- ✅ Отображение бронирований пользователя
- ✅ Навигация к профилю из header
