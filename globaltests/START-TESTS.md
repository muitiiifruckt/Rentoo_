# Как запустить тесты БЕЗ зависаний

## Важно: Сначала запустите серверы вручную!

1. **Запустите backend** (в отдельном терминале):
```bash
cd ../backend
venv\Scripts\activate
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

2. **Запустите frontend** (в другом терминале):
```bash
cd ../frontend
npm run dev
```

3. **Запустите тесты** (в третьем терминале):
```bash
cd globaltests
npm test
```

## Или используйте простой bat-файл:

Просто дважды кликните на `run-tests-simple.bat` - он запустит тесты и покажет результаты.

## Почему зависает?

Playwright пытается автоматически запустить серверы, и это может вызывать зависания в PowerShell. Лучше запускать серверы вручную.
