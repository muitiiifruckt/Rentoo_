# Исправления в тестах

## Исправленные проблемы:

1. **Ошибка с localStorage** - исправлена функция `clearStorage`, теперь она сначала переходит на страницу перед очисткой
2. **Ошибки в beforeEach** - все тесты теперь правильно очищают storage перед началом
3. **Селекторы** - улучшены селекторы для поиска элементов (добавлены `.first()`, улучшены ожидания)
4. **Ожидания** - добавлены правильные ожидания для загрузки страниц и элементов
5. **Таймауты** - увеличены таймауты для медленных операций

## Основные изменения:

- Все `clearStorage` заменены на прямую очистку после перехода на страницу
- Добавлены `waitForSelector` перед заполнением форм
- Улучшены селекторы для поиска элементов профиля и кнопок
- Увеличены таймауты с 5000 до 10000-15000 мс
- Добавлены дополнительные задержки для загрузки данных

## Запуск тестов:

```bash
npm test
```

Или используйте скрипт:
```bash
.\run-tests-simple.ps1
```
